# Installation Guide ## Install LXD First, install LXD using snap: ```bash sudo snap install lxd sudo lxd init --auto ``` Add your user to the LXD group: ```bash sudo usermod -a -G lxd $USER newgrp lxd ``` Verify installation: ```bash lxc --version lxc list ``` ## Clone Repositories Clone the main repository: ```bash git clone https://github.com/example/main-repo.git cd main-repo ``` Clone multiple repositories in parallel: ```bash mkdir projects && cd projects git clone https://github.com/example/repo1.git & git clone https://github.com/example/repo2.git & git clone https://github.com/example/repo3.git & wait ``` Clone a specific branch: ```bash git clone -b develop https://github.com/example/feature-repo.git ``` Clone with depth limit for faster download: ```bash git clone --depth 1 https://github.com/example/large-repo.git ```
